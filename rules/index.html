Basic Rules

üéØ The Objective

Fill the entire grid with numbers so that each row, column, and sub-grid contains all the numbers from 1 to the grid size exactly once. No number should be repeated in any row, column, or sub-grid.

üö´ What's Not Allowed

You cannot place the same number twice in any:

  1. Row (horizontal line)
  2. Column (vertical line)
  3. Sub-grid (the smaller rectangular sections)

Solving Strategies

üîç Scanning

Look for rows, columns, or sub-grids that are nearly complete. Often you can immediately see what number is missing.

üìù Process of Elimination

For each empty cell, determine which numbers are already used in its row, column, and sub-grid. The remaining numbers are possibilities.

üéØ Single Possibility

If a cell has only one possible number based on the elimination process, that's the answer for that cell.

üî¢ Hidden Singles

Sometimes a number can only go in one place within a row, column, or sub-grid, even if that cell has other possibilities.

Pro Tips

  • Start with the easiest cells first - look for rows, columns, or sub-grids with the most numbers already filled in.
  • Use pencil marks (or mental notes) to keep track of possible numbers for each cell.
  • Don't guess! Every move should be based on logical deduction.
  • Take breaks if you get stuck. Sometimes a fresh perspective helps you see new possibilities.
  • Practice regularly with easier puzzles before moving to harder ones.
  • Double-check your work frequently to catch mistakes early.
